# ClinicFlow MVP

## Vision

ClinicFlow is an AI-powered appointment booking platform for local clinics.

Patients should be able to:

- Book appointments
- Ask FAQs
- Contact clinic
- Chat on WhatsApp

Doctors should be able to:

- View appointments
- Cancel appointments
- Manage availability

---

## Features

### Website

- Home
- Services
- Doctors
- Contact
- FAQ
- Appointment Booking

---

### Appointment Booking

Patient enters:

- Name
- Phone Number
- Date
- Time

Appointment is stored.

---

### AI Receptionist

Can answer:

- Clinic timings
- Consultation fee
- Services
- Doctor availability

Can collect appointment information.

---

### Dashboard

Doctor can:

- View appointments
- Search patients
- Cancel appointment
- Update availability

---

### Integrations

- Google Calendar
- WhatsApp
- Google Sheets
